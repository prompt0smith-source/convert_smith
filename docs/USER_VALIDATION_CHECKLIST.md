# User Validation Checklist

이 체크리스트는 Convert Smith를 배포하거나 PDF 편집기 기능을 검증할 때 사용합니다.

## 설치 및 라이선스 고지

- 설치 마법사에서 약관 화면이 표시되는지 확인합니다.
- 약관에 동의하지 않으면 앱이 종료되는지 확인합니다.
- 설치 폴더에 `THIRD_PARTY_NOTICES.md`가 포함되어 있는지 확인합니다.
- 설치 폴더에 `legal/FFMPEG_SOURCE_OFFER.txt`가 포함되어 있는지 확인합니다.
- 설치 폴더에 `legal/PDFIUM_CHROMIUM_NOTICE.txt`가 포함되어 있는지 확인합니다.
- 설치 폴더에 `legal/licenses/electron/LICENSES.chromium.html`이 포함되어 있는지 확인합니다.
- 앱 하단에 `COPYRIGHT © JINKYU YOO`가 보이는지 확인합니다.

## 변환 기본 동작

- PNG/JPG/WEBP 최적화 후 전후 용량이 결과 보고서에 표시되는지 확인합니다.
- MP4 -> MP3 변환 후 MP3 파일이 정상 재생되는지 확인합니다.
- MOV -> MP4 또는 동영상 호환성 복구 후 일반 플레이어에서 재생되는지 확인합니다.
- JPG/PNG -> PDF 변환 후 각 이미지가 페이지로 들어가는지 확인합니다.
- PDF -> JPG/PNG 변환 후 글꼴과 배치가 원본과 다르게 깨지지 않는지 확인합니다.
- DOCX/XLSX -> PDF는 시작 전에 LibreOffice 상태 안내가 뜨는지 확인합니다.
- 원본 파일이 덮어써지지 않는지 확인합니다.

## PDF 도구

- PDF 병합 결과 파일명이 기본값으로 자연스럽게 생성되는지 확인합니다.
- PDF 분할 결과 파일이 `기존 파일명(1)`, `기존 파일명(2)` 형식으로 생성되는지 확인합니다.
- 우클릭 빠른 병합은 파일명 기준 순서로 병합되는지 확인합니다.
- 우클릭 빠른 분할은 앱 창을 불필요하게 띄우지 않고 결과를 생성하는지 확인합니다.

## PDF 편집기 뷰어

- PDF 편집기 진입 버튼이 기존 PDF 도구와 분리되어 있는지 확인합니다.
- PDF 편집기 창에는 PDF 자체가 중심으로 보이고, 하단 텍스트 목록 패널이 다시 생기지 않는지 확인합니다.
- 수정 모드가 아닐 때 PDF 텍스트를 마우스로 드래그 선택할 수 있는지 확인합니다.
- 한국어와 영어가 네모 문자로 깨지지 않는지 확인합니다.
- Chrome 또는 Acrobat에서 보는 원본과 기본 렌더링이 크게 다르지 않은지 확인합니다.

## PDF 편집기 텍스트 수정

- 수정 모드에서 텍스트를 클릭하면 해당 텍스트만 선택되는지 확인합니다.
- 입력 커서가 텍스트 영역에서는 I-beam으로 표시되는지 확인합니다.
- 텍스트를 수정하면 화면상 즉시 변경된 것처럼 보이는지 확인합니다.
- 빈 배경을 클릭하면 현재 수정값이 커밋되는지 확인합니다.
- Ctrl+Z, Ctrl+Y가 수정값 되돌리기와 앞으로 돌리기로 동작하는지 확인합니다.
- Ctrl+S가 저장으로 동작하는지 확인합니다.
- 삭제 버튼 또는 Delete 키를 누르면 선택 텍스트 삭제 확인 말풍선이 뜨는지 확인합니다.
- 저장 후 PDF를 다시 열어도 수정값이 되돌아가지 않는지 확인합니다.
- 수정한 텍스트가 기존 텍스트 위에 겹쳐 보이지 않는지 확인합니다.
- 흰 배경 마스킹이 표 선이나 주변 텍스트를 가리지 않는지 확인합니다.

## PDF 편집기 객체 인식

- 로고나 서명 이미지를 텍스트와 별도 객체로 인식하는지 확인합니다.
- 표 선이 텍스트와 별도로 하이라이트되는지 확인합니다.
- 표 내부 텍스트를 선택해도 표 선이 같이 움직이지 않는지 확인합니다.
- 이미지, 선, 표 후보를 인식하지 못한 경우 디버그 로그에 이유가 남는지 확인합니다.

## 실패 및 복구

- 손상된 PDF를 열었을 때 성공처럼 표시하지 않는지 확인합니다.
- 렌더링 실패 시 `logs` 폴더 또는 앱 디버그 로그에서 원인을 확인할 수 있는지 확인합니다.
- 변환 실패 후 임시 파일이 남지 않는지 확인합니다.
- 변환 중 앱을 종료해도 원본 파일이 변경되지 않는지 확인합니다.

## 배포 전 최종 명령

```bash
npm run build
npm run smoke
npm run dist
```

빌드된 설치 파일을 새 환경에서 설치한 뒤 위 항목을 다시 확인합니다.
